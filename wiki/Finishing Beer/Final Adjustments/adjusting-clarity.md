# Adjusting Clarity

*From Gordon Strong, "Brewing Better Beer" — Finishing Beer. Part of [[final-adjustments]]. The main treatment is in [[clarifying]] and [[stabilizing]].*

If finished beer is **still hazy**, or develops haze later, borrow mead techniques.

- **Fine with Super-Kleer K.C. or Sparkolloid** — meant for wine but work on beer, mead, and cider. Best used in bulk (carboy/keg), but small quantities work in soda bottles.
- **Stubborn haze:** Strong has had only one case where these didn't work — cleared with **bentonite first, then Sparkolloid** a few days later.
- These can take **hours to days**; **chilling helps**. Wait until clear, then **decant** into another container and **carbonate if needed**. Expect some **volume loss** — start with extra.
- **Filtering is a last resort** — it abuses the beer more than finings, strips flavor and color, and can introduce oxygen if done poorly.
