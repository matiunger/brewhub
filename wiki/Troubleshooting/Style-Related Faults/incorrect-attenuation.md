# Incorrect Attenuation

*From Gordon Strong, "Brewing Better Beer" — Troubleshooting. Part of [[style-related-faults]]. See also [[adjusting-final-gravity-and-attenuation]] and [[understanding-attenuation]].*

Attenuation sets how **dry or sweet** a beer tastes (and, less so, body). Errors hit **balance, drinkability, and style fidelity** — and because **dryness accentuates bitterness while sweetness masks it**, a change in attenuation shifts perceived bitterness **even at the same measured IBUs**.

> **Attenuation is driven more by wort composition than yeast strain** — look first at **grain selection and mashing**. Most yeast fully attenuate given proper conditions; less-flocculent strains and longer conditioning help a bit.

## Diagnose the Residual Extract

Taste what's left:

- **Sugary sweetness** → unfermented **sugars** remain → encourage continued fermentation.
- **Dry / full but not sweet** → **dextrins** (unfermentable by normal yeast) → add body without sweetness.

> Aim for **dextrins for body, not residual sugar** — sugar sweetness clashes with bitterness. Too many dextrins make it heavy and hurt drinkability. Some styles want residual sweetness; many are fully fermented.

## Fixes

| Want **more** attenuation (drier) | Want **less** attenuation (fuller/sweeter) |
|-----------------------------------|--------------------------------------------|
| Lower saccharification temp / step mash | Mash higher |
| Add sugar | Remove sugars |
| Lower the OG (lower FG, seems drier) | Raise the OG |
| | More crystal malts |
| | Add starchy adjuncts / dextrin malts |
| | Shorten conditioning |
