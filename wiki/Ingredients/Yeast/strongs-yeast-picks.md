# Strong's Yeast Picks

*From Gordon Strong, "Brewing Better Beer" — Mastering Ingredients ("So What Do I Do?"). Part of [[choosing-a-yeast-strain]].*

Working with a **smaller set** of strains lets you learn them deeply and get reproducible results. Strong keeps go-to strains but splits 10-gal batches to trial a new strain against a known one. He **avoids** sulfur-heavy lagers (WLP820 Oktoberfest), fusel/diacetyl producers (Ringwood), and most phenol producers.

## Go-To Strains by Style

| Style | Strain(s) | Notes |
|-------|-----------|-------|
| **English ales** | WLP002 / WY1968 | Fuller's strain, great for cask (very flocculent) — **don't drop below 65°F/18°C** or it quits. Ferments ~68°F (20°C). |
| English (malty/dark) | WY1318 London III | Mild, porter, old ale |
| English (dry / IPA) | WY1028 | |
| **Lagers** | WLP833 (bock) | Almost any style; ferment 48–50°F (9–10°C), lager 32–35°F (0–2°C) |
| Lager (neutral / high-attn) | WY2206 | |
| Lager (malty) | WY2124 | |
| Lager (warm fermentation) | WLP810 San Francisco | "Secret weapon" for Baltic porter |
| **Belgian (Trappist)** | WY3787 | From Westmalle (also used by Westvleteren); start cool 65°F (18°C), let rise; may need extra time + rousing for full attenuation |
| Belgian (variation) | WY1762, WY3522 (warm) | |
| Belgian (general workhorse) | WLP510 Bastogne / Orval | New favorite across many styles |
| **American ales** | WY1272 American II / WLP060 | A little extra fruitiness; stay 65–68°F (18–20°C) |
| American (neutral) | WY1056 / WLP001 | Absolutely clean |
| Scottish | WY1728 / WLP028 | |
| Kölsch | WY2565 / WLP029 | |
| Irish red / stout | WLP004 | |
| Belgian pale ale | WLP515 | |
| **Hefeweizen** | WY3068 | Favorite; WLP380 Hefe IV, WLP3638 for dark. Ferment below 65°F (18°C) |

> Strong personally avoids the **banana/solvent** character of WY1214 fermented warm (curious to try it at 60–62°F / 16–17°C). General rule: **match the yeast to the brewery/style first**, then confirm it has no show-stoppers.
